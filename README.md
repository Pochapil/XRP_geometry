# Программа для входных параметров моделирует наблюдаемые импульсы рентгеновских пульсаров

модель на каждом этапе проецирует элементы поверхности аккреционных колонн на плоскость наблюдателя, вычисляет излучаемую иненсивность в соответствии с излучением абсолютно черного тела и интегрирует по ним (учитывая эффекты затмений и ослаблений в оптически толстом веществе), чтобы получить изотропную светимость источника.

[скрипт](mayavi_plots/plot_mayavi.py) для визуализации, который позволяет отрисовать в 3D
